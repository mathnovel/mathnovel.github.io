## 纯色背景的作用

纯色的背景，其实是出于很多考量的


### 1. 提升转化率与用户体验
+ 消除视觉干扰：纯色背景能让顾客在极短的时间内（通常只有 0.5 秒）把注意力完全集中在产品的细节、颜色和形状上，减少背景杂色的心理干扰。

+ 视觉统一性：当搜索结果页面上的所有产品都是白底时，整个页面看起来整洁、专业，像一本高质量的目录，而不是凌乱的跳蚤市场。这增加了平台的信任感。

+ 准确传达信息：纯白背景最能还原产品的真实色彩（色彩对比度最高），减少因环境光反射导致的偏色，从而降低退货率。

### 2 技术与跨平台优势
+ 无缝集成：亚马逊的网页和 App 背景本身就是白色的。纯白背景的图片可以产生一种产品**“悬浮”**在页面上的视觉效果，与 UI 设计完美融合。

+ 移动端适配：在屏幕较小的手机上，复杂的背景会让产品显得更小且模糊。纯色背景有助于在低分辨率或小屏幕上清晰展现轮廓。

+ 后期灵活性：一旦拥有了一张高质量的纯色底图，后期通过 Photoshop 或 AI 工具更换背景、添加文案或制作合成海报的成本极低
  

### 3 心理暗示：专业与高端
在消费者的潜意识里，“白底图 = 官方/专业”，而**“实景图 = 个人闲置/非正规军”**。大品牌如苹果（Apple）、耐克（Nike）在官网上也大量使用纯色背景，这种视觉语言已经培养了用户的认知习惯，暗示产品质量更有保障。

### 过曝是否可行？

如果我们用超强的光源+白色遮光板，来作为背景，拍摄出的图片会出现一系列物理光学问题，不仅不会让背景更纯净，反而会毁掉产品的质感。

 
1. 边缘溢光与“雾化” (Light Wrap / Veiling Glare)
这是最常见的问题。当背景光强度远超正面光时，光线会像水一样“漫”过产品的边缘。

+ 现象：产品的轮廓变得模糊，边缘看起来像是半透明的，或者有一层白色的雾气。

+ 后果：原本锋利的金属或塑料边缘会消失，产品看起来像是在发光或融化。
2. 轮廓细节丢失 (Erosion of Detail)
对于细小的物体（如发丝、刷毛、精密零件的尖角），极强的背景光会产生衍射现象。

+ 现象：背景光会直接“吃掉”这些细小的结构。

+ 后果：如果你拍的是一件有细毛领的衣服，过强的背景光会让毛领看起来像秃了一块；如果是玻璃杯，杯壁的厚度感会完全消失。

3. 色彩冲淡与对比度下降 (Lens Flare)
过强的光进入镜头后，会在镜头内部产生多次反射。

+ 现象：整张照片的对比度剧烈下降，原本鲜艳的颜色变得灰蒙蒙的。

+ 后果：黑色产品会变成深灰色，彩色产品会失去饱和度，后期调色也很难还原那种“高级感”。

4. 出现“光晕” (Halation)
+ 现象：在产品和背景交界的地方，会出现一圈不自然的亮光。

+ 后果：这会让照片看起来非常有“廉价感”，像是早期低质量数码相机的拍摄效果 
<img src="/assets/img/1007.jpg" />

为什么要这样子做呢？

这里面涉及到ORBITVU的核心抠图技术点：**双重拍摄与蒙版技术 (Dual Shot / Masking)**

#### 双重拍摄与蒙版技术 (Dual Shot / Masking)

这是 Ortery（以及其他高端自动化摄影棚竞品）实现高质量、瞬间去背的“杀手锏”。对于透明、半透明或边缘复杂的物体 ，这种方法极其有效。

实现原理： 相机在极短的时间内（毫秒级）连续拍摄两张照片，但两次的灯光设置截然不同。

1. 第一张：产品照 (Product Shot)
+ 灯光： 正常的正面光、侧面光、顶部光打开，将产品照亮，展现细节和质感。
+ 目的： 获取用于最终展示的物体图像。

2. 第二张：剪影照/蒙版照 (Silhouette Shot)
+ 灯光： 关闭所有正面打光。只打开底部和背部的强力背光板。
+ 效果： 因为光线全部来自后方，不透明的产品会变成一个纯黑色的剪影，而背景则被照成纯白色。
+ 目的： 这张黑白照片直接生成了一个完美的“Alpha 蒙版 (Mask)”。黑色区域就是要保留的，白色区域就是要透明的。

3. 软件合成：
+ 软件瞬间将第二张图作为蒙版，应用到第一张产品图上，直接“切”出产品。
+ 优势： 速度极快，边缘极其精确，不需要复杂的 运算，基本是物理层面的精确。

扣除产品的轮廓以后，剩下的就是制作一张高亮纯色的背景图，将扣出来的产品贴上去，就完成高亮纯色背景的产品的制作了。


其他公司，比如PhotoRobot，也是运用了此类软硬件结合的方案来做自动化的抠图的，软件硬件缺一不可.

## 此类方案的优势VS劣势

优势已经在上面说明了，它的劣势呢？

+ 首先，一定要有要的光源，且要有足够多的光源，这一点决定了这种方案不可能便宜的
+ 自动化的时候，需要同个位置的2张照片在高度近似，这样才能用蒙板照来切图，也就是意味着需要上高精度的电机，这个方案复杂，不便宜
+ 需要使用单反相机来达到一定的拍摄效果，大家都懂：单反毁一生.
